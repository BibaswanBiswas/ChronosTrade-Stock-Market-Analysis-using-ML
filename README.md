# ChronosTrade-Stock-Market-Analysis-using-ML
In this WiDS project, we apply ML methods like ARIMA,LSTM and Sentiment Analysis to make stock price predictions
The Chronos_Week_1.ipynb has the code for Week 1 work, LSTM.ipynb is for Week 2 work and  Sentiment_analysis_and_Stock_market_prediction.ipynb is for the Week 3 and 4. The Report.pdf is the final report of the project.
